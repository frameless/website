# Onderzoek tijdens ontwikkeling

Wanneer wij een project voor je doen willen we natuurlijk dat hij voor alle gebruikers goed werkt. Daarom doen we standaard een onderzoek volgens de WCAG-EM methode van het W3C. Het onderzoeksrapport is bij ons onderdeel van de opdracht en wanneer je dat wil maken we er ook gewoon gelijk een toegankelijkheidsverklaring van.

## Mogen wij van WCAG-eend dat wel doen?

Volgens [DigiToeganklijk](https://www.digitoegankelijk.nl/toegankelijkheidsverklaring/onderzoek#zelf) wel, wij hebben de technische kennis, alle benodigde documentatie en het gereedschap in huis!
